TIPS

Raccourcis clavier:
  • INSERT - pour créer une nouvelle tache
  • TAB (lors de l'édition) - pour éditer la tache suivante
  • MAJ+TAB (lors de l'édition) - pour éditer la tache précédente
  • ESC - pour annuler l'édition de la tache
Cliquez ...
  • ... sur l'icone pour changer le statut de la tache
  • ... sur le bouton [X] pour supprimer une tache
  • ... sur le champs 'info' caché entre la description et le bouton de suppression
